# **App Name**: Recipe Snap

## Core Features:

- Ingredient Identifier: Image analysis using AI to identify ingredients and suggest potential recipes. Use a tool to decide if ingredients might be missing, or quantities adjusted
- Recipe Generation: Generate a recipe based on the identified ingredients, specified quantities, and time constraints.
- Recipe Display: Display the generated recipe with a list of ingredients, instructions, cooking time, and baking time.
- Recipe Adjustment: Allow users to adjust quantities, cooking time, and baking time, triggering a recipe recalculation.
- Image retrieval: Let users upload images via URL and local filesystem

## Style Guidelines:

- Primary color: Earthy green (#386641) for a natural, fresh feel.
- Secondary colors: Cream (#F0EAD6) and light brown (#DDB892).
- Accent: Terracotta (#BC6C25) for call-to-action buttons and highlights.
- Clean and readable typography that is easy to follow
- Use of simple, clear icons to represent ingredients, cooking times, and baking times.
- Clean, single-column layout, mobile-first layout, focused on the display of image and recipe details.
- Subtle animations when the recipe is generating.
