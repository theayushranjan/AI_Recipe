# 🍽️ AI Recipe Generator

AI Recipe Generator is a web-based application developed as part of a 4th semester Artificial Intelligence project. It generates personalized recipes based on user-provided ingredients, dietary preferences, and cuisine choices.

---

## 🚀 Features

* 🥗 **Ingredient-Based Recipe Generation**
  Get recipes using ingredients you already have.

* 🧠 **AI-Powered Suggestions**
  Smart recommendations based on user input.

* 🌍 **Cuisine Selection**
  Explore recipes from different cuisines.

* ⚙️ **Dietary Preferences**
  Supports vegetarian, vegan, and other dietary needs.

* 🎨 **User-Friendly Interface**
  Clean and intuitive UI for smooth interaction.

---

## 🛠️ Tech Stack

* **Frontend:** React, Next.js, Tailwind CSS
* **Backend:** Node.js
* **AI Integration:** Custom AI flows
* **Other Tools:** Firebase, TypeScript

## ⚡ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/theayushranjan/AI_Recipe.git
```

2. Navigate to the project directory:

```bash
cd AI_Recipe
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```

5. Open in browser:

```
http://localhost:3000
```
